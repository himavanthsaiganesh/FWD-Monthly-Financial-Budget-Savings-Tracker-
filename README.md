---------Monthly Financial Budget and Savings Tracker---------

The Monthly Financial Budget and Savings Tracker is a front-end web application developed using HTML, CSS, and JavaScript. The main objective of this project is to help users monitor and manage their monthly income and expenses in a simple and organized way.
In today's fast-paced world, many people struggle to track their spending habits and maintain proper financial planning. This application provides a structured system where users can record their income, track expenses across different categories, and calculate their monthly savings automatically.
By maintaining a clear financial record, the system helps users understand their spending behavior and encourages better budgeting practices.

Objectives -
The primary objectives of this project are:
To provide a simple system for managing monthly financial records
To help users track income and expenses efficiently
To calculate monthly savings automatically
To encourage responsible financial habits
To assist users in setting financial goals and planning future purchases

Features - 
The application includes several features that support financial tracking and budgeting:
Month selection for recording financial data
Monthly income entry
Addition of multiple expense categories with corresponding amounts
Automatic calculation of total expenses
Monthly savings calculation
Goal setting for purchasing items in the future
Tracking of total savings accumulated over time
Exporting financial data to an Excel file for documentation and analysis

System Workflow - 
The working process of the application follows these steps:
The user selects a month from the available options.
The user enters their monthly income.
The user adds expense categories along with the respective amounts.
The system calculates the total expenses.
The system determines the remaining savings for that month.
Users can set financial goals and track their savings progress.
The financial data can be exported to an Excel sheet.

Technologies Used - 
The project is developed using the following front-end technologies:
HTML – Used to structure the web application interface
CSS – Used to design and style the user interface
JavaScript – Used for implementing the application logic and calculations
SheetJS (XLSX Library) – Used for exporting financial data into Excel format
